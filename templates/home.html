Core Features

⚡ Real-Time Collaboration

Multiple users can collaborate simultaneously using WebSocket communication powered by Flask-SocketIO.

🏠 Room Based Architecture

Dedicated room isolation ensures secure and independent collaboration sessions.

💾 MySQL Persistence

Save and restore whiteboard state using a robust relational database backend.

🖼 Export Tools

Download whiteboards as PNG images or JSON files for future use.

✏️ Drawing Tools

Pen and eraser support with synchronized real-time updates across users.

🚀 Scalable Design

REST APIs, Socket.IO communication and modular Flask architecture for future expansion.
